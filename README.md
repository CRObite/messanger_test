## 🚀 Начало работы

### 🛆 Установка
1. Клонируйте репозиторий:
   ```sh
   git clone https://github.com/crobite/messanger_test.git
   ```
2. Перейдите в директорию проекта:
   ```sh
   cd messanger_test
   ```
3. Установите зависимости:
   ```sh
   flutter pub get
   ```

### ▶ Запуск приложения
Для запуска на Android:
```sh
flutter run
```
Для запуска на iOS:
```sh
cd ios && pod install && cd ..
flutter run
```
